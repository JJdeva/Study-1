# 1. TCP와 UDP의 차이점과 장단점

## A. TCP

- 연결형 프로토콜로 흐름제어와 오류제어를 함
- 데이터 송수신에 신뢰성이 있지만, 속도가 느림

## B. UDP

- 비연결형 프로토콜로 흐름제어, 오류제어를 하지 않음.
- 데이터 송수신에 신뢰성이 낮지만, 속도가 빠름
    - 동영상 스트리밍에 사용됨.
